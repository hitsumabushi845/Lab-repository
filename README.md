# Lab-repository
## 概要
　発光断面積計算に関する諸々のプログラムです．
　
- DataFile_to_CSVFile.py
- Matrix_to_List.py
- Input_A_coefficients.py
- graph.py

### DataFile_to_CSVFile.py
　電子捕獲断面積のデータ(Delimiter = whitespace)をCSVファイルに整形するプログラム
　
### Matrix_to_List.py
  隣接行列を隣接リストに変換するプログラム
  
### Input_A_coefficients.py
  アインシュタインのA係数を入力してCSVデータファイルを出力するプログラム
  
### graph.py
  グラフ探索のテストプログラム
